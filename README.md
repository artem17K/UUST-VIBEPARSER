# 📅 UUST Schedule VibeParser

![Python](https://img.shields.io/badge/python-3.10%2B-blue?style=for-the-badge&logo=python)
![Status](https://img.shields.io/badge/status-active-success?style=for-the-badge)

На 99% навайбкоденный парсер расписания для студентов **УУНиТ (UUST)**. Парсит расписание с портала `isu.uust.ru`, генерирует удобное изображение.

Даже это описание по большей части навайбкодило, мне стыдно такое выкладывать

Парсит не совсем идеально и надо улучшать, но мне лень, когда придётся тогда и улучшу

## ✨ Возможности

*   **Автоматический парсинг:** Загружает актуальное расписание вашей группы прямо с сайта вуза.
*   **Генерация изображений:** Создает аккуратную PNG-картинку с расписанием на неделю.
*   **Картинки котов:** Поддерживает автоматическую замену всего дня с военной кафедрой на **свою картинку** (на кота).
*   **Умная статистика:** Считает, сколько лекций и практик уже прошло, и сколько осталось до конца семестра (на основе вашего учебного плана).
*   **Отслеживание изменений:** При запуске сравнивает новое расписание с предыдущим сохранённым и сообщает об изменениях.
*   **Мигание пар:** Автоматически чередует сдвоенные пары по подгруппам (функция отключения пока не предусмотрена).

## 🚀 Установка и запуск

### 1. Подготовка окружения

Убедитесь, что у вас установлен [Python](https://www.python.org/downloads/). Затем установите необходимые библиотеки:

```bash
pip install requests beautifulsoup4 imgkit
```

### 2. Установка wkhtmltopdf (Обязательно!)

* Скрипт использует движок wkhtmltopdf для создания изображений. Без него генерация картинки работать не будет.

* Установите с [официального сайта](https://wkhtmltopdf.org/downloads.html).

* Запомните путь установки. Обычно это:
C:\Program Files\wkhtmltopdf\bin\wkhtmltoimage.exe

## ⚙️ Настройка

Откройте файл config.py в любом текстовом редакторе и отредактируйте блок настроек.

### 1. Путь к конвертеру

Укажите путь к файлу wkhtmltoimage.exe, который вы установили ранее.

```python
WKHTMLTOIMAGE_PATH = r"C:\Program Files\wkhtmltopdf\bin\wkhtmltoimage.exe"
```

### 2. Ваша группа
```python
GROUP_NAME = "ПИ-101Б"  # Укажите свою группу
```

### 3. Картинка поверх военной кафедры (Опционально)
Если хотите, чтобы день военной кафедры заменялся картинкой, укажите путь к файлу. 

Если хотите видеть обычное расписание — оставьте None.

Вариант с картинкой:
```python
MILITARY_DAY_IMAGE_PATH = r"C:\Users\User\Pictures\cat.jpg"
```

### 4. Счётчик остатка часов по плану (Опционально)

Заполните словарь PLANNED_CLASSES, чтобы скрипт мог считать оставшиеся пары.

```python
PLANNED_CLASSES = {
    "Базы данных": {"Лекция": 18, "Практика": 0, "Лабораторная": 18},
    "Интернет-программирование": {"Лекция": 9, "Практика": 0, "Лабораторная": 18},
    # ... добавьте свои предметы
}
```

## ▶️ Использование

Запустите скрипт через run.bat
